## Pattern: Expand Around Center

### When It Applies
- Palindromes
- Symmetry around center
- Need longest / count palindromic substring

### Key Insight
Every palindrome expands from center
Check all possible centers, odd and even

### Common Problems
- Longest palindromic substring
- Count palindromic substring
- Palindromic ranges

### Common Mistakes
- Forgetting even-length palindromes
- Using DP when O(1) space is enough
